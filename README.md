### Cool
#### About
The "Cool" mod introduces a range of new and convenient features to Minecraft, supporting the 1.21.4neoforge version. By pressing the C key in the game, players can obtain an item named "cool." Right-clicking while holding this item will open a graphical user interface (GUI). Within the GUI, players can click on different buttons to access a variety of practical effects, including instant healing, hunger recovery, jump boost, swiftness, health regeneration, strength, fire resistance, luck, poison, one-click suicide, water breathing, and slowness.

#### Download
- Modrinth: [Cool(酷)](https://modrinth.com/mod/cool-effect)
